## showcase:

to run you need to host esnya_food_backend from https://github.com/tadeohepperle/esnya-food-backend on localhost:8080.
To do this:

````bash
git clone https://github.com/tadeohepperle/esnya-food-backend
cd esnya-food-backend
dart run
````
